Features

Wee gives you a bootstrap of HTML, CSS, and JavaScript components which are simple to use and easily extendable.

  • Project organization

    Wee is organized in a simple, scalable hierarchy with a consistent structure for both style and scripts.

  • Mobile-first CSS

    With mobile traffic exceeding 50% globally its no longer a question of if but how. Wee answers that.

  • JavaScript toolset

    Wee's core JavaScript modules create logic to what can easily become a daisy-chained mess.

  • Build process

    Making everything compile, minify, and optimized isn't easy but Wee wraps it all in into a simple config file.

  • Static generator

    Originating from a need for a style guide generator this feature grew to be able to build full sites like this one.

  • Exceedingly scalable

    Wee powers tens of millions of pageviews each month and is trusted by fortune 500 websites.

  • Module extension

    Wee modules plug into both the build system and the Less for truly seamless integration with core features.

  • Legacy fallback

    Wee still handles IE8 both with ease and without compromising modern browser features.

  1. Agency inspired

    Created and maintained by the team at Caddis, Wee is fueled by solving real problems in real projects.

  2. Moving forward

    Not a day goes by we don't work to make Wee better. Our passion for front-end shows with every commit.

  3. Open source

    We're huge fans of open source. The Apache 2 License guarantees the freedom to use Wee without reservation.

  • Get started

    Ease into the basics. We'll walk you through downloading Wee and setting up your first project.

  • View source

    There's no better way to learn Wee than exploring the source. Dive into the code to learn more.
