---code---

Works the same as [Wee.$()](#core) but only returns the last result from the result set.

```html
<ul class="names">
	<li>John Doe</li>
	<li>John Smith</li>
	<li>Jane Doe</li>
	<li>Jane Smith</li>
</ul>
```

---code|modifier:split---

```javascript
Wee.$last('.names li');
```

```html
<li>Jane Smith</li>
```
