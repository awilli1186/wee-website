A handful of time-saving helper classes

Wee tries to stay out of your way and that means shipping with only the bare minimum of helper classes. As demonstrated, many of the helper classes can also be used as mixins.

Container #

Create a centered container based on @minWidth, @maxWidth, and @padContainer

.container {
    .content-box();
    .max-width(@maxWidth);
    .min-width(@minWidth);
    .check () when (@padContainer = true) {
        .padded();
    }
    .check();
}

.container();

display: block;
margin-left: auto;
margin-right: auto;
-moz-box-sizing: content-box;
-webkit-box-sizing: content-box;
box-sizing: content-box;
max-width: 1280px;
min-width: 260px;
padding-left: 6%;
padding-right: 6%;

Typography #

Style text to match predefined HTML element styling

Headings

The heading class will apply the general heading styling whereas the h1-h6 classes also apply the specific font size for that heading level.

.heading,
.h1,
.h2,
.h3,
.h4,
.h5,
.h6 {
    color: @headingColor;
    font-family: @headingFont;
    font-weight: @headingFontWeight;
    .line-height(@headingLineHeight);
    .spaced(@headingMarginBottom);
    small {
        font-weight: normal;
    }
}
.h1 {
    .font-size(@h1);
}
.h2 {
    .font-size(@h2);
}
.h3 {
    .font-size(@h3);
}
.h4 {
    .font-size(@h4);
}
.h5 {
    .font-size(@h5);
}
.h6 {
    .font-size(@h6);
}

Blockquote

.blockquote {
    color: @quoteColor;
    .border(left; @quoteBorderColor; @quoteBorderWidth);
    .font(@quoteFont; @quoteFontSize; @quoteFontWeight; @quoteLineHeight; @quoteFontStyle);
    .padding(@quotePaddingVertical; @quotePaddingHorizontal);
    cite {
        color: @citeColor;
        display: block;
        .font(@citeFont; @citeFontSize; @citeFontWeight; @citeLineHeight; @citeFontStyle);
        .margin(top; @citeMarginTop);
        &:before {
            .content(@citeIndicator);
        }
    }
    p {
        color: @quoteColor;
    }
}

Horizontal Rule

.hr {
    border: 0;
    display: block;
    height: 1px;
    .border(top; @ruleColor; @ruleHeight; @ruleStyle);
    .margin(vertical; @ruleMargin);
}

JS Hide #

Used by the core Wee script to toggle visibility

.js-hide {
    display: none !important;
}

Image Align #

Perfect for alignment of images in content blocks

Left

.img-left {
    float: left;
    .margin(.32em; @imageMarginSide; @imageMarginBottom; false);
}

.img-left();

float: left;
margin-top: 0.32em;
margin-right: 2rem;
margin-bottom: 2rem;

Right

.img-right {
    float: right;
    .margin(.32em; false; @imageMarginBottom; @imageMarginSide);
}

.img-right();

float: right;
margin-top: 0.32em;
margin-bottom: 2rem;
margin-left: 2rem;

Note: The img-left and img-right class names can be overridden in variables.less.

Not Printed #

If print styling is enabled exclude content from being printed

<div class="not-printed">This won't be output when printing.</div>
