Events →

Event binding to attach functionality to elements

Create organized interaction on your page with Wee’s simple event API. Support for mouseenter and mouseleave is baked in.

Bind #

Traverse DOM for available bindings

Deprecated: Wee.events.on() should be used instead using ‘ref:id’ selectors.

Bound #

Get currently bound events to optional specified element, event, and function

VariableTypeDefaultDescriptionRequired
selselection–Specific Wee selection
evtstring–Specific event name
fnfunction–Specific callback

By default bound will return all bound events.

Wee.events.bound();

[Object, Object, ...]

Element

Wee.events.bound('ref:element');

[Object, Object, ...]

Element Event

Wee.events.bound('ref:element', 'click');

[Object, Object, ...]

Fire #

Execute specific function by name and event

Deprecated: Wee.events.trigger() should be used instead using ‘ref:id’ selectors.

Map #

Add bindings to bound object with optional exec object and/or init boolean

Deprecated: Wee.events.on() should be used instead using ‘ref:id’ selectors.

Off #

Remove specified function to specified element and optional event and function

VariableTypeDefaultDescriptionRequired
selselection–Wee selection✓
astring–Event name or object of events
bcallback–Callback to remove

Target

If no event or callback is provided all element events will be removed.

Wee.events.off('ref:element');

Element Event

Wee.events.off('ref:element', 'click');

Element Event Callback

Wee.events.off('ref:element', 'click', function(e, el) {
    // Click logic
});

Multiple Elements

Wee.events.off({
    'ref:element': {
        mouseenter: function() {
            // Enter logic
        }
    },
    '.js-element': {
        click: function(e, el) {
            // Click logic
            e.preventDefault();
        }
    }
});

On #

Bind specified function to specified element and event

VariableTypeDefaultDescriptionRequired
selselection–Wee selection✓
astring, object–Event name or object of events✓
bcallback, object–Event callback or options object
cobject–Option parameters below
VariableTypeDefaultDescriptionRequired
argsarray–Callback arguments
contextselector, element–Wee selection context
delegateselector, element–Wee selection delegate
onebooleanfalseRemove the event after first execution
scopeobject–Callback scope

Simple

Wee.events.on('ref:element', 'click', function(e, el) {
    // Click logic
    e.preventDefault();
});

Once

Wee.events.on('ref:element', 'click', function(e, el) {
    // Click logic
    e.preventDefault();
}, {
    once: true
});

Delegation

Wee.events.on('.js-descendant', 'click', function(e, el) {
    // Click logic
    e.preventDefault();
}, {
    delegate: 'ref:element'
});

Multiple Events

Wee.events.on('ref:element', {
    click: function() {
        // Click logic
    },
    blur: function() {
        // Blur logic
    }
});

Multiple Targets

Wee.events.on({
    'ref:element': {
        mouseenter: function() {
            // Enter logic
        }
    },
    '.js-element': {
        click: function(e, el) {
            // Click logic
            e.preventDefault();
        }
    }
});

One #

Bind specified function to specified element and event for single execution

Deprecated: Wee.events.on() should be used with the one option set to true.

VariableTypeDefaultDescriptionRequired

astring–Event name✓
Wee.events.trigger('ref:element', 'click');
