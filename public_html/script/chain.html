Chain

Method chaining

Wee allows methods to be chained to selectors by omitting the first argument of the method. This often makes code more readable and permits multiple methods to be called on the same selection.

DOM #

In this example, the selector takes the place of the methodâ€™s first argument, and the second argument is passed in as the first.

$('.another-element').insertAfter('.element');

DOM traversal is made easy with chaining.

<ul>
    <li class="first"></li>
    <li class="second"></li>
    <li class="last"></li>
</ul>

$('.last').prev();

<li class="second"></li>

Events #

In this example, the selector is the element to which the event is bound.

$('.element').on('click', function() {
    // Callback logic
});

Here the event is being triggered on the selector.

$('.element').trigger('click');
