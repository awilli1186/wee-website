Data request and binding functions

Use Ajax to create dynamic experiences. You can submit any type of request with a number of callback options. Additionally, with view rendering you can parse JSON into templates from the response.

    }

Get & Parse

Retrieve JSON and immediately parse it into a template. No need to set the json parameter to true, it’s assumed.

{
    person: {
        firstName: 'Don',
        lastName: 'Draper'
    }
}

Wee.data.request({
    url: '/path/to/file.json',
    template: 'Hey person.firstName || Guest',
    success: function(parsed, raw) {
        console.log(parsed);
        console.log(raw);
    }
});

Post w/ Data

    method: 'post',

        password: 'secret'
    },
    success: function(data) {
        // Success logic
    }
