Features

Wee's HTML, CSS and JavaScript bootstrap components are simple to use and easily extensible.

  • Project organization

    Wee is organized in a simple, scalable hierarchy with a consistent structure for both style and scripts.

  • Mobile-first CSS

    With mobile traffic exceeding 50% globally its no longer a question of if but how. Wee answers the how.

  • JavaScript toolset

    Wee's core JavaScript toolset brings logic to what can often become a daisy-chained nightmare.

  • Build process

    Compiling, minifying, and optimizing everything can be a pain, but Wee manages it all with a simple config file.

  • Static generator

    Originating from a need for a style guide generator this feature grew to be able to build full sites like this one.

  • Exceedingly scalable

    Wee powers tens of millions of pageviews each month and is trusted by fortune 500 websites.

  • Module extension

    Wee modules plug into both the build process and the Less/JavaScript for seamless integration with core features.

  • Legacy fallback

    Wee still handles IE8 both with ease and without compromising modern browser features.

  1. Agency inspired

  2. Moving forward

    Not a day goes by we don't work to make Wee better. Our passion for front-end shows with every commit.

  3. Open source
