Features

  • Logical structure

    Wee is organized in a simple, scalable hierarchy with a consistent structure for both style and scripts.

  • JavaScript foundation

    Wee's core JavaScript library brings sanity to what can often become a daisy-chained nightmare.

  • Advanced scripting

    Built-in animation, asset loading, Ajax, event binding, HTML5 History, routing, and breakpoint syncing features.

  • Data binding

    Easily sync data models to the DOM using Wee's powerful templating to create dynamic, real-time interfaces.

  • Mobile-first CSS

    With mobile traffic exceeding 50% its no longer a question of building responsive but how. Wee answers that.

  • Build process

    Compiling, minifying, and optimizing can be painful, but Wee manages it all with a simple JSON config file.

  • Module extension

    Wee modules plug into both the build process and Less/JavaScript for seamless extensibility.

  • Static generator

    Originating from a need for a living style guide generator this feature evolved to build full sites like this one.

  • Get started

    Ease into the basics. We'll walk you through downloading Wee and setting up your first project.

  • View code

    There's no better way to learn Wee than exploring the source. Dive into the code to learn more.
