Features

  • Logical structure

    Wee is organized in a simple, scalable hierarchy with a consistent structure for both script and style.

  • Script foundation

    The core Wee JavaScript library brings sanity to what can often become a bloated, daisy-chained nightmare.

  • Advanced scripting

    Powerful animation, asset loading, Ajax, event binding, HTML5 History, routing, and responsive features.

  • Data binding

    Easily sync data models to the DOM using Wee's powerful templating to create dynamic, real-time interfaces.

  • Mobile-first CSS

    With mobile traffic exceeding 50% its no longer a question of building responsive but how. Wee answers that.

  • Automated building

    Compiling, minifying, and optimizing can be painful, but Wee manages it all with a simple JSON config file.

  • Modular extension

    Wee modules plug into both the build process and core JavaScript and Less for seamless extensibility.

  • Static generator

    Originating from a living style guide generator this feature evolved to power complex sites like this one.

  • Get started

    Ease into the basics. We'll walk you through downloading Wee and setting up your first project.

  • View code

    There's no better way to learn Wee than exploring the source. Dive into the code to learn more.
